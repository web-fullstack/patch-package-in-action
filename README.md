## dev

```sh
# cjs
$ yarn dev
$ yarn app

$ npx ts-node ./src/index.ts
$ npx ts-node ./src/app.ts


# esm
$ yarn dev-esm
$ yarn app-esm

$ npx ts-node-esm ./src/index.ts
$ npx ts-node-esm ./src/app.ts

```
